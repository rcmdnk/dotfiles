dotfiles
========

**Setting files for bash**

# Installation

Below commands will make links to dotfiles in $HOME directory
If there is a file, it makes backup file (.bak)

    cd ~/tmp
    git clone git@github.com:rcmdnk/dotfiles
    cd dotfiles
    ./install.sh
